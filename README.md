## 🚀 **Getting Started**

### **Prerequisites**
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for external fonts and EmailJS)

### **Local Development**
```bash
# Clone the repository
git clone https://github.com/ReyhanehAYousefi/reyhanehayousefi.github.io.git

# Navigate to project directory
cd reyhanehayousefi.github.io

# Open in browser
open index.html
# OR serve with a local server
python -m http.server 8000
# OR use Live Server extension in VS Code
```

### **Deployment**
The website is automatically deployed via GitHub Pages when changes are pushed to the main branch.
